# Changelog

## [1.0.0-rc1] — 2026-05-24

### Added
- **Hub Veil Sanctum** — meta progression, soul shards, 12 permanent upgrades
- **3 boss fights** — Warden (F2), Veil Serpent (F3), Heart of Abyss (F3 finale)
- **20 unique enemy AI overlays** + elite affixes (fast, tank, splitter, vampiric, volatile)
- **Attack telegraphs** — visual warnings for dangerous attacks
- **Room modifiers** — darkness, poison fog, frenzy, gilded, elite wave
- **3 biomes** — catacombs, fungal, crystal (floor-based tints)
- **Juice** — screen shake, hitstop, floating damage numbers
- **Touch controls** — virtual joystick + dodge/attack buttons (Android-ready)
- **Endless mode** — floors 4–6 with scaling
- **Daily challenge** — seeded run per calendar day
- **Codex, synergies, affixes** (Fase 2 carry-over)
- Play Store docs: privacy policy, store listing template

### Changed
- Main scene is now Hub; runs load `Run.tscn`
- Game over returns to Sanctum with soul shard rewards

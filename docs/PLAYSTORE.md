# Play Store — Checklist (Fase 6)

## Persiapan

1. **Godot Android export templates** — download versi sama dengan editor
2. **JDK 17** + Android SDK (Android Studio)
3. Edit `export_presets.cfg` — keystore path (jangan commit keystore)
4. Build AAB: Project → Export → Android

## Full description (EN)

You are a Veil Warden — trapped in a dungeon that reshapes every run.

**Veil of Abyss** is a solo-dev action roguelike built for 15–30 minute sessions with long-term depth.

- 100 items with lore & unique effects
- 20 enemies with distinct attack patterns
- 3 epic bosses + endless mode
- Tag synergies, affixes, codex
- Permanent upgrades (Soul Shards) — no pay-to-win
- Touch-ready controls

Every run is a new journey. Die? Return to the Sanctum, learn, grow stronger.
